## 核心工具库使用说明

- **文件位置**：`MCP/tool.md`
- **服务对象**：执行与落地阶段代理，用于在调度任务时快速定位可用工具。
- **调用提示**：优先按“工具编号”或“简短别名”检索；下方提供每个工具的适配问题类型、使用方式与典型示例。

### A. 计算与执行工具

#### `MCP.code_interpreter` (T1)
- **适配问题类型**：`code_generation`、`data_analysis`
- **工具说明**：在受控沙箱中生成并执行 Python 代码，可用于脚本编写、调试验证、数据处理与可视化等任务。支持即时查看执行结果，便于快速迭代。
- **文档操作能力**：可运行预置的 finish_form 同步脚本，直接在协作表 Markdown 中插入/更新阶段输出（自动加入 `AUTO_SYNC` 标记，避免覆盖手工内容）。
- **典型示例**：根据用户提供的数据样本生成清洗脚本，并输出统计汇总；复现算法片段验证推导正确性；或在阶段完成后调用同步脚本，自动将结果写入 `finish_form/*.md`。

---

### B. 信息检索工具

#### `MCP.tavily` (T2)
- **适配问题类型**：`information_retrieval`、`fact_verification`
- **工具说明**：调用 Tavily 在线搜索服务获取最新网页信息，聚焦事实核验、资讯追踪与资料查找。自动返回候选链接与摘要，支持引用来源。
- **典型示例**：核对新闻事件的时间线并给出引用网址；或搜集行业数据报告以支持研究综述结论。
